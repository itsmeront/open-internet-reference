# Intake

This directory contains unverified candidates extracted from imported project materials.

Intake files are working queues. They are not verified knowledge pages, contact records, or bibliography records. Candidates should move out of intake only after they are checked against primary or high-confidence sources and given proper metadata.

## Current Queues

- `candidate-topics.md`: possible knowledge topics and claim areas.
- `candidate-entities.md`: possible organizations, people, and entity records.
- `verification-queue.md`: prioritized verification tasks.
