# Bibliography

This directory contains source records used by OIR.

Source records should be specific enough for another researcher to find and verify the same material. Prefer official and primary sources where available.

## Categories

- `academic/`: academic papers and scholarly sources.
- `imported/`: imported project materials and internal drafts.
- `legal/`: cases, statutes, regulations, and legal reference sources.
- `organizations/`: official or high-confidence organization pages.
- `technical/`: RFCs, standards, specifications, and canonical technical materials.
