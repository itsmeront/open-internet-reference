# Handbook

The handbook provides a curated, linear presentation of OIR material for readers who want a structured overview instead of a reference index.

Each generated handbook edition includes:

- An **About OIR** section explaining the project and research standards
- An **Index** of included knowledge records
- A **Glossary** of record definitions
- The main **Knowledge Records** content
- A **Sources Referenced** appendix for cited primary sources

The full OIR bibliography is maintained separately because it catalogs every source record in the repository, including material not cited in the handbook.

The current generated prototype is available at [Generated Handbook](../generated/handbook.md).

A browser-printable version is available at [Print Handbook](../generated/print-handbook.md). It includes a cover page with the OIR logo, release label, generation date, and edition metadata. Use your browser's print dialog to save it as a PDF.
