# Source Intake

Imported project files are tracked as source-intake material, not as verified factual authority.

## Current Intake Status

OIR has intake records for the initial bootstrap prompt, project charter, roadmap, Axona reference manual draft, and outreach contact lists.

Unverified candidates extracted from those materials are tracked in `intake/`.

## Rules

- Internal drafts may guide research direction.
- Substantive claims require independent verification.
- Contact details require official-source verification before use.
- Candidate claims and entities should be separated from verified facts.
