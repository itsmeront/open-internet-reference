# Bibliography

Bibliography records will track primary and secondary sources used by OIR, including legal authorities, standards, academic papers, books, official publications, archival records, and reputable journalism.

Each source should have a stable identifier and enough metadata for independent verification.

The current generated source index is available at [Generated Bibliography](../generated/bibliography.md).

The generated source-usage index is available at [Citation Index](../generated/citations.md).

## Current Seed Categories

- Technical standards and RFCs
- Legal cases and statutes
- Official organization pages
- Academic papers
- Imported project materials
